## 2.1　唯識の起源

唯識思想は、紀元4世紀頃のインドで体系化されました。中心人物は無著（むじゃく、Asanga）と世親（せしん、Vasubandhu）の兄弟です。二人が属した学派は「瑜伽行派」（ゆがぎょうは、Yogacara）と呼ばれ、大乗仏教の二大学派の一つに数えられます。

仏教はもともと「苦しみの原因を理解し、苦しみを滅する」ことを目指す思想です。仏教では、苦の原因は無明・渇愛・煩悩・業など、複数の層で語られてきました。唯識はこの探求を、**認識（識）の構造を精密に分析する**という方向で深めた学派です。識の働き方を分析することで、無明や煩悩、業の蓄積がどのように苦を生むかを示そうとしました。「苦の原因は識にある」と単純化してしまうと心理学の話に寄りすぎますが、唯識が問題にしたのは、苦が生じる**構造**を識の側から精密に解明することだった、と捉えてください。

本書は仏教学の書物ではありませんので、唯識思想の全体を詳述することはしません。本章では、以降の章で必要になる唯識の基本的な枠組みを、ソフトウェアエンジニアのための全体像として提示します。

### この章の読み方

この章には用語がたくさん出てきます。しかし、**この章で暗記する必要はありません**。各概念は対応する章（第3章以降）で繰り返し登場し、そこで丁寧に解説します。

最初に押さえてほしいのは、次の四つだけです。

1. 唯識は人間の認識を**八つの識**に分けて分析する。
2. 八つの識は機能の違いで区別されており、第七・第八識は通常の意識からは直接観察しにくい。「深層心理」のような階層図と完全に重ねないこと。
3. すべての経験は第八識（**阿頼耶識**）に「種子」として蓄積され、新しい経験の出方を条件づける。
4. 同じ一つのものごとが現れるときの**三つの相**（思い込み／条件依存／ありのまま）を区別する。

この章は地図です。各章を読みながら、全体の位置を確認するために使ってください。

## 2.2　八識 — 認識の八つの層

唯識は、人間の認識を八つの「識」に分けて分析します。「識」とは認識の働きのことです。八つは機能の違いで区別されており、互いに影響し合いながら動いています。本書では、感覚的な前五識から始めて、概念処理の第六意識、自己執着の末那識、種子を蔵する阿頼耶識へと順に紹介します。便宜上「表層／深層」という言葉を使うこともありますが、これは観察のしやすさの順序を指す言葉であって、現代心理学の「深層心理」とは別の枠組みです。

第1章で「識の転変（pariṇāma）の中に主客が現れる」と述べました。世親は『唯識三十頌』でこの転変を三つのレベルに整理しています。**異熟転変**（いじゅくてんぺん、阿頼耶識のレベル）、**思量転変**（しりょうてんぺん、末那識のレベル）、**了別境転変**（りょうべつきょうてんぺん、前六識のレベル）の三つです。以下に紹介する八識は、この三層の転変に含まれる形で働いています。第9章で扱う四分説（相分・見分・自証分・証自証分）は、それぞれの識の転変がどのような内部構造を持つかを精密に分析したものです。三種転変の用語そのものを暗記する必要はありませんが、「識の転変」という語が出てきたとき、それは八識のうちどこかのレベルで起きている活動を指している、と理解してください。

### 前五識（ぜんごしき）— 五つの感覚的な認識

最も表層にある五つの識です。注意していただきたいのは、前五識は感覚器官（眼・耳・鼻・舌・身）そのものではない、ということです。唯識では、感覚器官のことを「根」（こん）と呼び、識とは区別しています。認識が成立するには、根（器官）・境（対象）・識（認識の働き）の三つが揃う必要があります。眼識とは「眼という器官を通じて色（しき、視覚的な対象）に触れたときに生じる認識の働き」のことです。

| 識 | 対象 |
|----|------|
| 眼識（げんしき） | 色（しき）— 色彩や形 |
| 耳識（にしき） | 声（しょう）— 音 |
| 鼻識（びしき） | 香（こう）— 匂い |
| 舌識（ぜつしき） | 味（み）— 味わい |
| 身識（しんしき） | 触（そく）— 触れた感覚 |

前五識は対象を直接的に了別します。この直接的な知覚のことを「現量」（げんりょう）と言います。しかし前五識は、**概念を通じた判断はしません**。眼識は赤い色を知覚しますが、それが「リンゴだ」とは知りません。「リンゴだ」と概念的に判断するのは、次に紹介する第六意識の仕事です。

→ 第3章で詳述

### 第六意識（だいろくいしき）— 概念を形成する心

日常語で「意識」と呼んでいるものに最も近い識です。

この識の主な働きは**分別**（ぶんべつ、vikalpa）です。対象を概念によって区別し、カテゴリに分け、名前を付ける認識の働きです。「これはリンゴだ」「あれは机だ」「この人は顧客だ」。こうした概念的な判断はすべて第六意識の分別です。

第六意識は二つの形で働きます。一つは前五識と同時に働く場合（**五倶意識**）で、感覚的な知覚に概念的な判断を加えます。赤い色を眼識が知覚すると同時に、第六意識が「リンゴだ」と判断する、という具合です。もう一つは前五識から独立して働く場合（**独頭意識**）で、想像、思考、記憶の想起、夢などがこれにあたります。人間の知的活動の大部分は、この識が担っています。

→ 第4章で詳述

### 末那識（まなしき）— 「自分」を掴み続ける識

第七識です。前五識や第六意識に比べると馴染みのない概念ですが、唯識の体系において決定的に重要な役割を果たしています。

末那識を理解するには、まず次のような問いから始めるとよいでしょう。われわれは意識的に「自分」について考えていないときでも、あらゆる経験を「自分の」経験として受け取っています。景色を見るとき「自分が見ている」、音を聞くとき「自分が聞いている」。いちいち意識しなくても、すべての認識に「自分の」という感覚が貼りついています。この感覚はどこから来るのでしょうか。

唯識の答えは、末那識です。末那識は、次に説明する阿頼耶識（第八識）の見分（けんぶん、阿頼耶識の認識する側の働き。詳しくは第9章で扱います）を対象として掴み、それを「自己」だと見なし続ける識です。「見なし続ける」と言ったのは、末那識が一瞬も休まず働いているからです。第六意識は深い眠り（無夢の睡眠）・気絶・無想定・滅尽定といった限られた状態で一時的に停止しますが（夢は独頭意識の働きなので、夢を見ている間は第六意識が活動しています）、末那識はこれらの状態でも原則として働き続けます。ただし、玄奘訳『成唯識論』系の標準説では、滅尽定においては末那識のうち染汚末那（煩悩を伴う自己執着の部分）は止まるとされ、末那識全体が完全に働き続けるわけではありません。本書ではこの詳細には立ち入らず、末那識は「通常の状態では途切れない」という点を押さえるだけにします。

この「自己だと見なす」働きは、四つの根本煩悩を常に伴っています。

| 煩悩 | 内容 |
|------|------|
| 我癡（がち） | 阿頼耶識の本当の性質（無我であること）を知らない |
| 我見（がけん） | 阿頼耶識を固定的な「自己」だと見なす |
| 我慢（がまん） | その「自己」に基づいて自他を比較する |
| 我愛（があい） | その「自己」に愛着する |

末那識が厄介なのは、第六意識の判断に影響を与えるにもかかわらず、第六意識からは直接観察できないことです。第六意識が何かを判断するとき、末那識はその背後で「自分にとってどうか」という色づけを加えています。しかし第六意識の側からは、その色づけが末那識から来ていることが見えません。自分の判断が中立だと思っているのに、実は「自分」を軸にした偏りがかかっている。末那識はそのような構造を生み出す識です。

→ 第5章で詳述

### 阿頼耶識（あらやしき）— すべてを蔵する流れ

第八識であり、八識の最深層です。「阿頼耶」はサンスクリット語で「蔵（くら）」を意味します。

唯識は難解な思想として知られていますが、その難しさの中心にあるのが阿頼耶識です。難解さの正体は、「固定した記憶庫」のようなイメージで近づいてしまうと阿頼耶識を取り逃がしてしまう、という点にあります。流れとして捉える、というのが最初の難所です。

象徴的な逸話を一つだけ紹介します。三島由紀夫が『豊饒の海』第三巻『暁の寺』の執筆にあたって唯識を学んでいたとき、インド学者の松山俊太郎に唯識について尋ねたところ、松山はそれが常人に近づきがたい思想だとした上で、「水のなかに一段一段下りていく階段があって、知らない間に足まで水がきて、知らない間に胸まできて、知らない間に溺れている」ようなものだと形容したと伝えられています。澁澤龍彦が『三島由紀夫おぼえがき』（原著：中央公論社、1983年／中公文庫、1986年）で伝聞として記したエピソードであり、本書もその伝聞に拠っています。一次発言の出典は未確認です。

ここで強調したいのは「やはり唯識は近寄れない思想だ」ということではなく、「固定した像で掴みに行くと滑り落ちる」という難所の感触です。本書では、この流れとしての阿頼耶識を、ソフトウェアの言葉で順を追って捉え直していきます。

#### なぜ阿頼耶識という概念が必要なのか

阿頼耶識は、いくつかの哲学的な難問に答えるために導入されました。

まず、**記憶と習慣の問題**です。昔の経験を今思い出すことができるのは、なぜでしょうか。昨日学んだことが今日の行動に影響するのは、なぜでしょうか。前五識も第六意識も現在の認識しか扱えません。過去の経験を保持し続ける場所が必要です。

次に、**意識が途切れる問題**です。深い眠りや気絶の間、第六意識は止まっています。しかし目が覚めると、眠る前の記憶は残っています。意識が途切れている間も、経験の蓄積を保持し続けるものがなければなりません。

さらに、**行為と結果の連続性の問題**です。仏教は因果（行為が結果を生む）を重視しますが、行為が結果として実るまでに時間がかかることがあります。その間、行為の影響力を保持するものが必要です。

阿頼耶識は、これらの問題に対する唯識の回答です。

#### 阿頼耶識の性質

阿頼耶識は、過去のあらゆる経験を**種子**（しゅうじ、bīja）として蓄えています。種子とは、行為や経験が残す潜在的な力のことです。この種子は、条件が揃うと**現行**（げんぎょう、現在の具体的な認識や行為）として発現します。そして現行は、また新たな種子を阿頼耶識に残します。

阿頼耶識それ自体は**無覆無記**（むぶくむき）です。煩悩に覆われておらず（無覆）、善でも悪でもない（無記）という意味です。阿頼耶識はいわば中立的な蔵であり、善い種子も悪い種子も分け隔てなく蔵します。善悪の方向性を持つのは、蔵されている個々の種子の側です。

#### 容器ではなく流れ

ここで注意すべき点があります。「蔵」と聞くと、物を入れておく固定的な容器を想像するかもしれません。しかし阿頼耶識は、静的な容器とはまったく異なります。

唯識ではこれを「恒転如暴流」（ごうてんにょぼうる）と形容します。「常に転じること暴れる流れの如し」。激流のように絶え間なく流れ続ける、ということです。種子は絶え間なく生滅し、新たな種子が薫習され、古い種子が現行として発現します。この流れが止まることなく続いている。阿頼耶識とは、この流れそのもののことです。

川にたとえるなら、川底に堆積した土砂（種子）が水流の条件次第で巻き上がり（現行）、巻き上がった土砂がやがてまた沈殿する（薫習）。この動的な過程の全体が阿頼耶識です。

#### 個人を超えた次元

ここまでの説明は、阿頼耶識を「個人の経験を保持する装置」として描いてきました。しかしそれだけでは、阿頼耶識の深さの半分も伝えられていません。

阿頼耶識に蔵される種子には、二種類あります。自分自身の行為や経験が残す「不共種子」（個別の種子）と、複数の有情が共通に持つ「共相種子」です。共相種子が各々の阿頼耶識に蔵されているからこそ、異なる存在が「同じ山」「同じ川」を経験できます。あなたと私が同じ世界を共有できるのは、各々の阿頼耶識に共通の種子が植わっているからです。阿頼耶識は個人の記憶庫であると同時に、複数の存在が世界を共有する根拠でもあります。

次に、時間的なスケールです。阿頼耶識に蔵される種子は「現在の個人の経験」だけではありません。唯識は「無始以来」（むしいらい）という表現を用います。始まりなき過去から積み重ねられてきた種子が蔵されているという考え方です。阿頼耶識の時間的スケールは、個人の一生を遥かに超えています。

さらに根本的な点があります。阿頼耶識は、他の識からその正体を把握されません。末那識は阿頼耶識を対象として取りますが、それを「自己」と誤認するだけで、阿頼耶識の本来の姿を知りません。第六意識にいたっては、通常は阿頼耶識の存在に気づくことすらできません。認識の地盤でありながら、認識によって正確には捉えられない。そういう構造になっています。

これら三つの次元を合わせると、阿頼耶識の位置づけが変わります。それは個人の記憶や習慣を保持する入れ物ではなく、個人の経験が現れてくる場そのもの、複数の存在が世界を共有する根拠です。

#### 末那識との関係

先に述べたように、末那識はこの阿頼耶識を「自己」と見なし続けます。激流のように流れ続けるものに、固定した自己を見出してしまう。唯識における自我意識とは、この構造のことです。

## 2.3　三性説 — ものごとが現れる三つの相

唯識は、あらゆるものを三つの性質（三性、trisvabhāva）から捉えます。これは三つの「もの」が別々に存在するという意味ではなく、**同じ一つのものごとが現れるときの、依存関係に基づく三つの相**を区別する枠組みです。「主観的な見方の違い」ではありません。後で詳しく扱うように、依他起性（条件によって生起している側面）が基盤にあり、そこに執着が重なれば遍計所執性として現れ、執着が離れれば円成実性として現れます。本書ではこの三相を、ソフトウェア設計におけるモデルの捉え方を整理する手がかりとして読み替えていきます。

### 遍計所執性（へんげしょしゅうしょう）— 思い込みの見方

「あまねく（遍く）計らい（計）、執着するところ（所執）の性質（性）」。名前の通り、言葉によって概念を固定化し、実体として執着してしまう見方のことです。

たとえば「顧客」という概念を定義したとき、あたかも「顧客」という確固たる実体が世界に存在するかのように思い込む。これが遍計所執性です。

### 依他起性（えたきしょう）— 関係の見方

「他に依って起こる性質」。すべてのものは他の条件に依存して生じているという見方です。独立した実体はなく、条件と条件の関係の網の中にある。仏教の根本原理である「縁起」（えんぎ、ものごとは原因と条件によって生じること）と通じる考え方です。

### 円成実性（えんじょうじっしょう）— ありのままの見方

「円かに成就した真実の性質」。遍計所執性（思い込み）を離れて、依他起性（関係の網）をありのままに見たときに現れる、もののありのままの姿です。

三つの性質の関係を整理すると、こうなります。まず、ものごとは条件の関係の中で生じている（依他起性）。しかしわれわれは、その関係の網の一部を切り取って実体視してしまう（遍計所執性）。この実体視を手放して、関係の網をそのまま見たとき、ものの本来の姿が見える（円成実性）。円成実性は依他起性とは別の何かではなく、依他起性から遍計所執性を取り除いたものです。

## 2.4　種子と薫習 — 因果のメカニズム

### 種子（bija）

阿頼耶識に蔵される潜在的な力です。過去の行為や経験の結果として植えられ、条件が揃うと現行（現在の認識や行為）として発現します。

### 薫習（くんじゅう、vasana）

「香りを染み込ませる」という意味の言葉です。衣服に香を焚きしめると、香そのものがなくなっても香りが残ります。これと同じように、行為や経験が終わった後も、その影響が阿頼耶識に種子として残ることを薫習と呼びます。

良い行為は良い種子を植え、悪い行為は悪い種子を植えます。そして蓄積された種子が、将来の現行（認識や行為）の方向を決めていきます。

### 名言薫習（みょうごんくんじゅう）

薫習の中でも特に重要なものが名言薫習です。言語を繰り返し使うことで、その言語が認識に刻印を残し、世界の見え方そのものを形成していく、という考え方です。使う言葉が変われば、世界の見え方も変わります。

→ 第8章で詳述

## 2.5　四分説 — 識の内部構造

識の内部がどのような構造になっているかを分析した理論です。一つの識の中に、四つの側面（分）があるとします。

| 分 | 働き | 日常的なたとえ |
|----|------|--------------|
| 相分（そうぶん） | 認識される側面。対象の像 | 鏡に映った姿 |
| 見分（けんぶん） | 認識する側面。認識の働き | 鏡が映す力 |
| 自証分（じしょうぶん） | 認識が起きていることの自覚 | 「今、鏡を見ている」という自覚 |
| 証自証分（しょうじしょうぶん） | 自覚の自覚 | その自覚が正しいことの裏づけ |

ここで重要なのは、四つがすべて**一つの識の内部**にあるという点です。「認識する主体」と「認識される対象」は、識の外に別々に存在するのではなく、識の活動の中に同時に現れます。これは第1章で述べた「識の転変の中に主客が現れる」という構造を、より精密に分析したものです。

→ 第9章で詳述

## 2.6　転識得智 — 識が智慧に変わる

唯識の修行の究極的な目標は「転識得智」（てんじきとくち、識を転じて智を得ること）です。

煩悩に染まった識が、修行を通じて質的に転換し、清浄な智慧になります。識の働きそのものが消えるのではなく、その**性質**が根本から変わるのです。八識はそれぞれ対応する智に転じます。

| 識 | 智 | 転換の意味 |
|----|-----|-----------|
| 前五識 | 成所作智（じょうしょさち） | 感覚が、適切な行為を成し遂げる智慧に変わる |
| 第六意識 | 妙観察智（みょうかんざっち） | 概念的な分別が、ありのままに観る智慧に変わる |
| 末那識 | 平等性智（びょうどうしょうち） | 自我への執着が、自他を平等に観る智慧に変わる |
| 阿頼耶識 | 大円鏡智（だいえんきょうち） | 種子の蔵が、一切をそのまま映す鏡の智慧に変わる |

## 2.7　本書の地図

以上の概念群の関係を整理します。

```
識の構造（第3章〜第6章）
┌─────────────────────────────────────────┐
│  前五識 ──→ 第六意識 ──→ 末那識         │
│  (感覚)    (分別)     (執着)           │
│       └──────────┬──────────┘           │
│                  ↕                      │
│            阿頼耶識                     │
│         (種子の蔵)                      │
└─────────────────────────────────────────┘

識の分析（第7章〜第9章）
┌─────────────────────────────────────────┐
│  三性説: 思い込み / 関係 / ありのまま     │
│  種子説: 種子 ←→ 現行（薫習の循環）     │
│  四分説: 対象 / 認識 / 自覚 / 自覚の自覚  │
└─────────────────────────────────────────┘

識の転換（第10章〜第12章）
┌─────────────────────────────────────────┐
│  転識得智:                               │
│  前五識 → 成所作智  末那識 → 平等性智    │
│  意識  → 妙観察智  阿頼耶識→ 大円鏡智    │
└─────────────────────────────────────────┘
```

本書はこの地図に沿って進みます。第3章〜第6章で識の構造を確認し、第7章〜第9章でその性質を分析し、第10章〜第12章で識の転換を論じます。各章は唯識の概念の解説から始め、ソフトウェア設計への展開を行います。
